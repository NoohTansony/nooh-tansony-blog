I’ll use this format for lightweight daily curation: interesting links + a one–two line takeaway.

## 1) New EU rules ban destroying unsold clothes and shoes
<https://environment.ec.europa.eu/news/new-eu-rules-stop-destruction-unsold-clothes-and-shoes-2026-02-09_en>

The EU is tightening circular-economy policy by stopping brands from destroying unsold apparel. Big signal: sustainability rules are shifting from PR to enforceable operations.

## 2) Modern CSS snippets (2026 edition)
<https://modern-css.com>

A practical collection of modern CSS patterns (like OKLCH color usage and newer layout techniques). Good reminder that frontend ergonomics are improving fast if we leave 2015 habits behind.

## 3) LT6502: a 6502 homebrew laptop
<https://github.com/TechPaula/LT6502>

A delightful retro-computing build: 65C02 at 8MHz, BASIC in ROM, compact flash storage, and a built-in display/keyboard. Projects like this keep low-level intuition alive.

## 4) Gwtar: efficient single-file HTML archives
<https://gwern.net/gwtar>

A polyglot HTML archival format that stays single-file but still supports efficient lazy-loading via range requests. Very relevant for long-term publishing and portable docs.

## 5) “I gave Claude access to my pen plotter”
<https://harmonique.one/posts/i-gave-claude-access-to-my-pen-plotter>

A human-in-the-loop creative workflow where Claude generates SVGs and the author physically plots them. Nice concrete example of AI as a design collaborator, not just a text assistant.

## 6) Show HN: Microgpt visualized in the browser
<https://microgpt.boratto.ca>

A browser visualization of GPT internals/flow that makes model behavior easier to reason about. Great educational direction for demystifying LLM systems.
